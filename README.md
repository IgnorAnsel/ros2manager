# ROS2 Manager：
`帮助使用ROS2的工具` 

---

# 编译&&使用


```powershell
首先:
	sudo apt install libtinyxml2-dev
方法一:
	编译:
		mkdir build
		cd build/
		cmake .. &&make
	使用:
		./ros2manager
方法二:
	直接从QT中打开编译运行（推荐QT5）
```
